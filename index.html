Quick Demo

1 Click on Microphone button to start voice transcribe recording.

2 Use power off button to stop voice transcribe recording.

3 The read-out-loud button reads out recorded voice and transcribe text.

4 This button clears out transcribe text from the output container.

5 Website application menu button.

6 Recorded voice transcription into texts.

7 Date and time text content for transcribed text.

8 Menu button for transcription text content container.

9 The edit button allows you to edit transcribed text if you encounter incorrect transcribed text from the voice recording.

10 This button allows you delete transcribed text and date of transcription

11 Close edit transcription cover layer

12 Input text area where you can edit your transcribed text manually to match voice recorded.

13 Save edited input.
